## 📌 Sobre o projeto

Aplicação web para gerenciamento completo de clientes, desenvolvida como projeto acadêmico na UTFPR. Permite cadastrar, visualizar, editar e remover clientes, com persistência de dados em JSON.

---

## ✨ Funcionalidades

- Cadastro de novos clientes
- Listagem de clientes
- Edição de dados
- Remoção de clientes

---

## 🛠️ Tecnologias

- HTML5 (Boilerplate) + Bootstrap
- CSS3
- Node.js + Express.js
- JSON (persistência de dados)

---

## 🚀 Como rodar

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/cadastro-clientes-utfpr

# Instale as dependências
npm install

# Inicie o servidor (comando a definir)
```
